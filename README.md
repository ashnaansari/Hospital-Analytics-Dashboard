# 🏥 Hospital Dashboard (Power BI)

## 📌 Overview
This project is a **Hospital Dashboard** built using **Power BI**. It provides insights into hospital operations like patient data, admissions, billing, and overall performance.

The dashboard helps in making better decisions by showing data in a simple and visual format.

---

## 🎯 Objectives
- Analyze patient records and hospital performance  
- Track admissions and discharges  
- Monitor revenue and billing  
- Support data-driven decision making  

---

## 📊 Features
- Interactive dashboards  
- Patient details and demographics  
- Department-wise analysis  
- Revenue and billing insights  
- Time-based trends (daily, monthly, yearly)  

---

## 🛠️ Tools & Technologies
- Power BI Desktop  
- Power Query  
- DAX (Data Analysis Expressions)  
- Data Visualization  

---

## 📂 Project File
- `Hospital dashboard2.pbix` → Main dashboard file  

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open it in Power BI Desktop  
3. Use filters and slicers to explore data  


---

## 📌 Key Insights
- Patient admission trends  
- Revenue distribution  
- Department performance  
- Peak hospital activity time
